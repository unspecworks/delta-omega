# DELTA OMEGA

A portable ultra-low-profile (ULP) wireless 3Ã—5+2 split keyboard.


## Overview

- Switch support: Cherry MX ULP or Kailh PG1316s
- 34 keys, splayed and staggered with choc spacing
- Wireless
- CNC aluminum case
